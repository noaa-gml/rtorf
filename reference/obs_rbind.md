# rbind obspack

return rbind obs data.tables

## Usage

``` r
obs_rbind(dt1, dt2, verbose = TRUE)
```

## Arguments

- dt1:

  first data.table

- dt2:

  second data.table

- verbose:

  logical to show more information

## Value

numeric vector
