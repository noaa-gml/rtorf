# return numeric vector in intervals

return numeric vector in intervals

## Usage

``` r
obs_freq(x, freq, ...)
```

## Arguments

- x:

  numeric, longer than \`freq\`

- freq:

  numeric vector

- ...:

  findInterval arguments

## Value

numeric vector
